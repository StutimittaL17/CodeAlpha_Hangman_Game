🎮 Hangman Game

📖 Description

This is a simple text-based Hangman game built using Python. The player guesses letters to reveal a hidden word.

🚀 Features

- Random word selection
- Limited attempts (6)
- Console-based gameplay

▶️ How to Run

python hangman.py

🛠️ Technologies Used

- Python
